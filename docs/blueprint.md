# **App Name**: Nexus

## Core Features:

- Conversational SQL Agent: A natural language tool that interprets user intent to generate and execute accurate SQL queries against industrial databases using AI reasoning.
- Live Database Connector: Secure middleware to link external SQL sources and provide real-time visibility into machine downtime and maintenance logs.
- Unified Auth Center: Firebase-powered authentication supporting email, Google, and GitHub login with state-aware navigation flows.
- AI Insight Tables: Clean, tabular visualization of raw query results formatted for corporate scannability and readability.
- Chat Session Repository: Persistent storage for conversational history using Firestore to allow users to revisit previous maintenance analyses.
- Schema Insight Tool: An intelligent tool that uses AI reasoning to analyze database metadata and recommend relevant prompts based on table relationships.
- Minimalist Gemini Workspace: A split-pane interface with a floating, auto-growing input bar for a distraction-free data analysis experience.

## Style Guidelines:

- Primary typography and borders: Deep Executive Slate (#1e293b); Main surface: Pure white (#FFFFFF); Sidebars: Frost white (#ffffff).
- Accent: Polished Cobalt (#9603ff) for buttons, active badges, and key calls-to-action.
- Font: 'Inter' with semi-bold headers and regular body text weights for an objective, machined aesthetic.
- Icons: 'lucide-react' with a consistent 1.5px stroke weight in monochrome slate.
- Layout: Split-pane design with a 280px collapsible frost-textured sidebar and a wide-centered conversational stream.
- Animations: Smooth ease-in-out width transitions for sidebar toggles and elegant fade-ins for message appearances.
